## HOW TO USE THIS DOCUMENT

Every row is an ingredient page on Supplifyed.
Every ingredient page is a funnel into the three-tier product stack.

**Evidence ratings:**
- `Strong` — 3+ independent peer-reviewed studies, consistent findings
- `Moderate` — Evidence exists, some inconsistency or limited study size
- `Mixed` — Studies exist but findings conflict
- `Limited` — Few or no independent studies

**Priority:**
- `Phase 1` — Build manually. First 10. Learn every step.
- `Phase 2` — Automation assisted.
- `Phase 3` — Fully automated.
