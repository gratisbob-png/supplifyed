# SPEC 04 — AFFILIATE LAYER
## Link Generation, Commission Tracking, Network Rules

---

## Three Links Per Product

Every product displays up to three links:

| Link | Network | When Shown |
|------|---------|-----------|
| Official Store | Brand direct / Awin / CJ / ShareASale | When brand has affiliate link |
| Amazon | Amazon Associates | Always |
| Generic ↓ | Amazon Associates (search) | Always for supplements; not for devices |

## Link Generation

All links generated by `src/lib/affiliate.ts`.

Amazon branded: `https://www.amazon.co.uk/dp/{ASIN}?tag={AMAZON_PARTNER_TAG}`
Amazon generic: `https://www.amazon.co.uk/s?k={encoded_ingredient_name}&tag={AMAZON_PARTNER_TAG}`

## Amazon Associates Rules (Non-Negotiable)

- RRP only — never live pricing, never a dynamic price endpoint
- Tag must be present on every Amazon link
- "As an Amazon Associate I earn from qualifying purchases" in footer

## Network Priority

| Network | Commission Rate | Products |
|---------|----------------|---------|
| Amazon Associates | 1-5% | All products |
| Awin | 10-25% | Brands on Awin network |
| CJ Affiliate | 10-25% | Brands on CJ network |
| ShareASale | 10-25% | Brands on ShareASale |

## Link Checking

All links checked weekly via `npm run links:check`. Dead links flagged immediately, link_status updated to 'dead', link removed from page until fixed.

The generic ingredient link is never removed even if brand link is dead — it connects to the Amazon search for the raw compound and always works.

## The Generic Link Rule

The generic link is the fourth data point that sits beneath all tiers. It shows — without saying — that the branded product is largely a known compound with a markup. This link cannot be removed by manufacturer submission. It is non-negotiable.
